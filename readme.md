
#### 实现功能
本系统是一套课堂管理的网站。首先系统提供用户注册和登录功能。有学生端和教师段两个不同的入口:
- 对于学生端，可提供加入特定的课程，提交该课程的作业，下载该课程老师上传的课件。
- 对于教师端。可发布新的课程，并对已有的课程中的学生进行管理。学生的信息、提交作业的情况均有可视化的数据统计。
